# Personal Introduction Program

## Project Overview
This beginner Python project collects basic information from the user and displays a personalized welcome message.

### Objectives
- Learn `input()`
- Store data in variables
- Display formatted output using f-strings
- Write and run a complete Python program

## Project Structure
```
README.md
personal_intro.py
requirements.txt
screenshot.png
```

## Setup Instructions
1. Install Python 3.10 or later.
2. Open a terminal in the project folder.
3. Run:
   ```bash
   python personal_intro.py
   ```

## Features
- Asks for Name
- Asks for Age
- Asks for City
- Asks for Favorite Hobby
- Displays a friendly welcome message

## Code Structure
- `input()` for user input
- Variables to store values
- f-strings for formatted output
- `print()` for display

## Testing
Example input:
- Name: Alex
- Age: 21
- City: Mumbai
- Hobby: Coding

Expected output:
A personalized welcome message containing all entered information.
