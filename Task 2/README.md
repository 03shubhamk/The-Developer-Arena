# 🎓 Student Grade Calculator

A beginner-friendly Python project that calculates student grades based on marks entered by the user.

This project is designed for beginners learning:

- Variables
- User Input
- If-Else Statements
- While Loops
- Functions
- Input Validation
- Python Programming Basics

---

# 📌 Features

✅ Accept Student Name

✅ Accept Student Marks

✅ Input Validation (Marks must be between 0 and 100)

✅ Automatic Grade Calculation

✅ Encouraging Messages

✅ Functions

✅ While Loop for Invalid Input

---

# 📝 Grading Logic

| Marks | Grade |
|--------|-------|
| 90-100 | A |
| 80-89 | B |
| 70-79 | C |
| 60-69 | D |
| 0-59 | F |
